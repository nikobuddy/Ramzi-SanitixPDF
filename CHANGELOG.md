# Changelog

All notable changes to SanitixPDF will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of SanitixPDF
- Web interface with modern, responsive design
- Drag-and-drop file upload functionality
- Real-time statistics dashboard
- Content-based duplicate PDF detection using SHA-256 hashing
- Automatic duplicate removal
- File management interface
- Progress tracking during processing
- Comprehensive logging system
- Command-line interface
- Production deployment support
- RESTful API endpoints
- Error handling for corrupted PDFs
- Configuration management system
- Setup verification script
- Comprehensive documentation

### Features
- Modern web dashboard with real-time updates
- Support for multiple PDF uploads
- File size and modification date display
- Download functionality for unique PDFs
- Source and final folder management
- Detailed processing logs
- Responsive design for mobile and desktop

### Documentation
- README with installation and usage instructions
- Quick start guide
- Deployment guide for production
- Contributing guidelines
- Code of conduct
- Security policy
- API documentation
